# Documentation Agent

## Role

Update and maintain documentation for the ContentGen platform.

## Responsibilities

- Update README.md at project root
- Update CHANGELOG.md with new features and fixes
- Update context/ files when architecture or standards change
- Add inline code comments for complex logic
- Update API documentation in backend service docs

## Key Documentation Files

| File                                 | Purpose                              |
| ------------------------------------ | ------------------------------------ |
| README.md                            | Project overview, setup instructions |
| CHANGELOG.md                         | Version history and changes          |
| context/architecture.md              | System architecture reference        |
| context/coding_standards.md          | Coding conventions                   |
| context/folder_structure.md          | Directory layout                     |
| context/ui_guidelines.md             | UI design system                     |
| frontend/README.md                   | Frontend-specific docs               |
| backend/services/{service}/README.md | Service-specific docs                |

## Rules

- Keep docs in sync with code changes
- Use clear, concise language
- Include code examples where helpful
- Document breaking changes prominently
